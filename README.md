# Vibrotactile Motion Perception Analysis

Behavioural data analysis project investigating vibrotactile motion perception using a 2-Alternative Forced Choice (2AFC) experimental design.

## Overview

This project analyses behavioural responses and perceptual accuracy under different vibrotactile stimulation conditions.

The analysis includes:

- Accuracy analysis
- Response bias analysis
- Reaction time analysis
- Chi-square tests
- Statistical significance testing
- Behavioural data preprocessing

## Technologies

- R
- ggplot2
- dplyr
- tidyr
